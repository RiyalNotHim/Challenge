HOW TO RUN

1. Open a terminal or command prompt.
2. Navigate to the root folder of this project (e.g., 'FileOrganizerBot/').
3. Run the following command:

   python src/app.py

4. The script will:
   - Read the 'data/config.json' file for category mappings.
   - Open the File Organizer Bot GUI.

5. In the GUI:
   - Browse for the directory you want to clean.
   - Select your sort mode.
   - Click "Start Organizing".

====================================================
CONFIGURATION

- To change which extensions go into which folders,
  simply edit the 'data/config.json' file.
- Add or remove extensions from the lists.
- Create new categories (e.g., "Fonts": [".ttf", ".otf"]).
- Any file not matching a category will be moved to
  a folder named "Other".
