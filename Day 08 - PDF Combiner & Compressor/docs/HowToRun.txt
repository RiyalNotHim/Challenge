HOW TO RUN

1. Open a terminal or command prompt.
2. Navigate to the root folder of this project (e.g., 'PDF_Utility/').
3. **Install the required dependencies:**

   pip install -r requirements.txt

4. Run the application with the following command:

   python src/app.py

5. The script will:
   - Open the PDF Utility GUI.
   - You can now use the "Merge" or "Compress" tabs.

====================================================
NOTES

- Requires Python 3 and the 'PyMuPDF' library.
- All file operations are run in a background thread
  so the GUI does not freeze.
